Kindselektoren

Nachfahrenselektor vs. Kindselektor

Die erste Listenebene soll eingerückt werden, die zweite nicht.

nav#zwei > ul { padding-left: 0; } nav#eins ul { padding-left: 0; } nav a { color: #333; text-decoration: none; }

Der Nachfahrenselektor

Mit nav ul { padding-left: 0; } werden beide Listenebenen eingerückt.

  • Startseite
  • Reisen
    • Italien
    • Island
  • Kontakt

Der Kindselektor

Mit nav > ul { padding-left: 0; } wird nur die erste Listenebene eingerückt.

  • Startseite
  • Reisen
    • Italien
    • Island
  • Kontakt
