!important

Mit vorsicht einsetzen. Beispiel JQuery Theme:
Es gibt den standard Jquery style ui-corner-top
Wenn man genau diesen Style umbedingt ändern will macht man in seiner css einen Eintrag:
#meinelement ui-corner-top:border-radios:25 !important;
Das bedeutet das ich den Style nur für Elemente mit der klasse ui-conrer-top anpasse welche sich innerhalb des Elementes mit der id "meinelement" befinden. So kann ich auch Framework css verändern ohne das allgemeine Theme zu zerstören.
