Das Element mark

Das Element mark funktioniert wie ein Textmarker, dient aber dazu Text zu kennzeichnen, der für den Benutzer besonders relevant ist. mark wird also nicht wie strong oder em vom Autoren in den Text geschrieben, sondern ist meist das Ergebnis einer Interaktion mit der Website.
Zum Beispiel für eine Volltextsuche wenn gefundene Wörter schon markiert werden.
