Elemente

<div> Element dient Hauptsächlich zum unterteilen der Webseite in verschiedene Bereiche.
<table> Strukturierte Darstellung
<ul> Listen Menüs
<header> Definiert den Kopfteil ("header") einer Seite oder eines Abschnitts. Er enthält oft ein Logo, den Titel der Website und die Seitennavigation.
<footer> Definiert den Fußteil ("footer") einer Seite oder eines Abschnitts. Er enthält oft Copyright-Hinweise, einen Link auf das Impressum oder Kontaktadressen. (SEO ignore)
<nav> Beschreibt einen Abschnitt der ausschließlich Navigationslinks enthält. (zum Beispiel topMenu,SEO ignore)
<article> Beschreibt eigenständigen Inhalt, der unabhängig von den übrigen Inhalten sein kann. Artikel immer dann wenn man ohne den rest der Seite auch weiss um was es geht.
<aside> Zusätzliche Informationen zum Inhalt. Ohne aside sollte der Inhalt trotzdem verständlich sein.
<address> Kontaktinformationen in einem Address tag. (SEO good)
<main> definiert den Hauptteil der seite. Nur einmal pro Seite. (Meist der Teil wo informationen stehen ohne Menu ohne footer Header, SEO good)
